# TaskFlow Client

This is the React frontend for TaskFlow.

## Getting Started

1.  **Install Dependencies**:
    Navigate to this directory (`TO--DO-APP/frontend`) and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    This will start the local development server (usually at http://localhost:5173).

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

-   `src/components`: Reusable UI components (Layout, TaskModal, etc.)
-   `src/pages`: Page components (Home, Login, Dashboard, etc.)
-   `src/context`: State management using React Context (Auth, Tasks)
-   `src/index.css`: Global styles
-   `src/App.jsx`: Main application component and routing
